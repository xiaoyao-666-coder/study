A real-time irrigation scheduling tool has been developed for maize fields across the contiguous United States (CONUS). By integrating subseasonal to seasonal (S2S) climate forecast ensembles with the Soil-Water-Atmosphere-Plant (SWAP) model, this tool uses stochastic optimization to schedule irrigation on any day of the growing season, balancing water costs, crop yields and net profit. A human-computer interaction framework is designed to provide real-time irrigation recommendations while incorporating farmer feedback. Various state-of-the-art CONUS-scale driving datasets (such as soil hydraulic parameters) are used to reduce reliance on in-situ observations and extend the tool’s applicability to diverse field conditions across CONUS. The package includes the source code, SWAP model input data, and usage instructions. This tool has been developed within the framework of the Dashboard for Agricultural Water Use and Nutrient Management (DAWN) project and is also accessible via the DAWN dashboard: https://dawn.umd.edu/.

The tool can run on Linux systems or be adapted for Windows with minor path modifications. If users want to use it in Windows, the codes should be modified to use relative paths or Windows-compatible paths, and the SWAP model executable file (swap.exe) should be used to replace swap_test (refer to the SWAP model documentation: https://www.swap.alterra.nl/).

To use this tool, execute Main_win.py, which calls various functions from the Maize folder. Additionally, Main_win_ensemble_mean.py is provided, allowing users to directly use the forecast ensemble mean to represent future climate conditions. This script includes detailed annotations and explanations.

Users can adjust parameters such as field location (by modifying latitude and longitude), water price, crop price, crop growth parameters, the current date of use, sowing date, and past irrigation applications. When using the tool in a new location for the first time, historical data is typically required to calibrate parameters. The data folder includes historical datasets such as ERA5-Land climate data and leaf area index to support parameter calibration.

For each simulation, the predicted results are saved in the Maize folder. Files named result_forec (including .irg, .crp, .inc files, etc.) contain the predicted results from SWAP.
day_scheduled represents the forecasted irrigation schedule, calculated as the mean across different ensemble predictions. all_day_ir_var_results.csv contains irrigation depths under different climate forecast ensembles.
