# GEFS QDM 日分配向 raw 收缩交叉验证 v1

## 背景

因果当前周期 QDM 加七日水量保持改善了 CRPS、Brier 和集合覆盖率，但逐日集合均值 MAE/RMSE 变差。稳定性审计表明，误差恶化主要来自数量最多的常规缩放组，而不是少数极端缩放组，因此不采用缩放硬截断。

## 冻结变换

对每个成员和预报日定义：

`P_alpha = P_raw + alpha * (P_qdm_vp - P_raw)`。

候选固定为 `alpha = 0.25, 0.50, 0.75, 1.00`。`alpha=0` 是 raw 基线，不作为订正候选。由于 `P_raw` 与 `P_qdm_vp` 的成员七日总量相同，所有候选仍严格保持 raw GEFS 七日总量。

## 数据边界

- 只使用已经生成的 2015-2018 因果 OOF 预测；
- 不重新拟合 QDM；
- 不使用 2019 或 2024 做本轮拟合、候选选择或评分。

## Gate

在既有 gate 基础上新增代理模型输入相关门槛：

- 汇总逐日集合均值 MAE 不劣于 raw；
- 汇总逐日集合均值 RMSE 不劣于 raw；
- 汇总七日 MAE、CRPS、平均 Brier 不劣于 raw；
- 重事件覆盖率不能两项同时下降，发生频率误差不劣于 raw；
- 七日 MAE、CRPS、Brier 各至少 3/4 年不劣于 raw；
- 无负值或非有限值，每成员七日总量误差不超过 `1e-8 mm`。

若多个候选通过，按汇总 CRPS、Brier、逐日集合均值 MAE、较小 alpha 的顺序预先锁定一个候选。通过者才允许进入 2019 验证。
