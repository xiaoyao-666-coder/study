# GEFS 七日累计两阶段线性缩放交叉验证 v1

## 论文依据

Shah 与 Mishra（2016）先将 GEFS 集合均值聚合为 1-7 天累计降水，再对训练期 90 分位以上周事件估计极端缩放因子，随后估计总体周降水缩放因子，并采用留一年交叉验证。本实验将该方法适配到当前 5 站点、5 成员、7 天 GEFS reforecast 数据。

## 冻结候选

- `weekly_two_stage_linear_global`
- `weekly_two_stage_linear_site_only`

只允许全局和按站点两种空间分组。

## 两阶段算法

对每个站点-起报周期，先计算各成员七日总量，再取成员平均得到周期集合均值七日总量 `S_gefs`；观测七日总量记为 `S_obs`。

1. 在训练组中计算 `q90 = quantile(S_gefs, 0.9)`。
2. 对 `S_gefs > q90` 的训练周期计算 `c_ext = sum(S_obs) / sum(S_gefs)`。
3. 先令训练周期 `S_stage1 = c_ext * S_gefs`（极端）或 `S_gefs`（非极端）。
4. 计算总体因子 `c_all = sum(S_obs) / sum(S_stage1)`。
5. 目标周期最终因子为 `c_all*c_ext`（超过训练 q90）或 `c_all`（未超过 q90）。
6. 同一站点-周期所有成员、所有日乘同一最终因子。

该变换保持原始七天内时序、零降水位置和成员排序，但允许订正七日总量。

## 与论文的适配

论文按给定年内日期和网格点拟合。当前最早折每个站点、每个固定日期只有 15 年，90 分位以上约 1-2 个样本。按照老师已确认的全局/按站点分组，本版在 6-8 月六个起报日期内池化，不增加日期或季节分组。最早折全局约 450 个站点-周期样本，按站点约 90 个样本。

## OOF 划分

2019 和 2024 不得用于拟合、选择或本轮评分。

## Gate

- 汇总逐日集合均值 MAE、RMSE 不劣于 raw；
- 汇总七日 MAE、CRPS、平均 Brier 不劣于 raw；
- 重事件覆盖率不能两项同时下降，发生频率误差不劣于 raw；
- 七日 MAE、CRPS、Brier 各至少 3/4 年不劣于 raw；
- 因子、订正降水均为有限非负值；
- 拟合年不得包含验证年、2019 或 2024。

若两个候选均通过，按汇总七日 MAE、逐日 RMSE、CRPS、Brier、候选名称顺序锁定一个候选进入 2019 验证。
