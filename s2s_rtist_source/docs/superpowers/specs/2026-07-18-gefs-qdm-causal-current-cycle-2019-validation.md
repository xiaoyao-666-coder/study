# GEFS QDM 因果当前周期目标 CDF 2019 验证 v1

## 冻结候选

唯一候选为 `qdm_global_current_cycle_7d_volume_preserving`。该候选由 2015-2018 因果 OOF gate 选出，2019 阶段不得改变目标 CDF、QDM 分组、水量保持公式、零值策略、缩放上限或 gate。

## 拟合与应用

- 历史 CDF：2000-2018 GEFS 与有效 GHCN-D 主站记录。
- 每个 2019 起报周期独立构造目标 CDF，只使用当时可见的 5 站点、5 成员、7 天预报，共 175 个 GEFS 值。
- 后续起报周期及 2019 GHCN-D 观测不得进入目标 CDF。
- 每个站点-周期-成员的订正后 7 天总量严格保持 raw GEFS 总量。
- 不增加事后缩放上限，继续报告 QDM 相对变化、缩放倍数和 raw 回退次数。

## 证据边界

2019 起报日期此前用于过探索性分析，因此本轮用于验证因果实现与跨年方向一致性，不能称为全新独立留出验证。2024 不得用于本轮拟合、选择或评分。

## Gate

沿用冻结规则：7 天 MAE、CRPS、平均 Brier、重事件覆盖率和发生频率均不劣于 raw；无负值或非有限值；每成员 7 天总量误差不超过 `1e-8 mm`；全部六个目标 CDF 批次必须为 175 行且未来周期使用数为 0。
