# GEFS QDM 水量缩放稳定性审计 v1

## 目的

因果当前周期 QDM 在 2015-2018 OOF 与 2019 探索验证中改善了 CRPS、Brier 和集合覆盖率，但水量保持步骤出现较大缩放倍数。本审计判断这些高倍数组是否罕见、是否集中制造逐日误差，以及概率增益是否可能依赖不稳定的日分配。

## 输入

- 冻结的 2015-2018 因果 OOF 完整周期预测与成员水量审计；
- 冻结的 2019 因果验证完整周期预测与成员水量审计。

不重新拟合 QDM，不改变预测，不使用 2024。

## 统计

- 缩放区间：raw 回退、`<=2`、`(2,5]`、`(5,10]`、`(10,20]`、`>20`；
- 阈值计数：`>5`、`>10`、`>20`；
- 每成员 7 天逐日 MAE 的 raw、订正值及差值；
- 每个区间的组数、占比、改善比例和最大日降水；
- 缩放倍数最高的事件明细。

本轮是描述性审计，不新增事后 gate。审计结果用于决定是否设计 raw/QDM 日分配收缩，不直接晋级 2024。
