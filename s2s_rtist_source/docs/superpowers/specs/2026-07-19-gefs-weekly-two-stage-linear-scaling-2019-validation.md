# GEFS 按站点七日累计两阶段线性缩放 2019 验证 v1

## 冻结候选

唯一候选为 `weekly_two_stage_linear_site_only`。该候选由 2015-2018 扩展窗口 OOF 的预锁定 gate 选出；2019 阶段不得改变空间分组、90 分位阈值、两阶段公式、时间池化或 gate。

## 拟合与应用

- 使用 2000-2018 的 GEFS 与有效 GHCN-D 完整七日站点-周期拟合；
- 每个站点独立拟合训练期 90 分位、极端因子和总体因子；
- 2019 GHCN-D 只用于评分，不参与因子拟合；
- 目标周期只使用自身 GEFS 集合均值七日总量判断是否进入极端区间；
- 同一站点-周期全部成员和全部七天使用同一冻结因子；
- 不使用未来 2019 周期，不使用 2024。

## Gate

- 七日 MAE、逐日集合均值 MAE、逐日集合均值 RMSE 不劣于 raw；
- CRPS、平均 Brier 不劣于 raw；
- 重事件覆盖率不能两项同时下降，发生频率误差不劣于 raw；
- 因子和订正降水均为有限非负值；
- 因子拟合中 2019 观测使用数必须为 0。

2019 日期此前用于探索性方法分析，因此结果属于验证集证据，不是全新独立留出证据。2024 仍保留为最终测试。
