# GEFS 冻结周尺度订正 2024 GHCN-D 参考一致性审计

## 目的

前一项五周期诊断使用 gridMET，而训练和验证使用 GHCN-D。本审计在不改变冻结
artifact、候选或 alpha 的前提下，用地面站记录重新评分相同五周期，判断参考产品变化
是否影响结论。

## 两种证据口径

1. `primary_station_available_cycles`：只使用 2000-2019 拟合时的同一主站，并只评价
   七天记录完整的站点-周期。这是严格的同站点参考，但 P1、P4 主站在目标日期不可用。
2. `complete_fallback_station_all_sites`：主站 35/35 完整时继续使用主站，否则使用已下载
   候选中距离最近且 35/35 完整的站点。该口径覆盖全部五站点，但不是严格同一主站。

替代站只根据预先核查的覆盖率和距离选择，不查看 raw 或订正预报误差。两种口径都属于
已经探索过的五周期诊断，不能声称是完全独立最终测试，也不得用于调 alpha。

## 不确定性

以起报日期为块，对五个周期进行 10000 次有放回重采样，分别报告七天 MAE、CRPS 和
Brier 的订正减 raw 差值及 95% bootstrap 区间。
